이 게임은 1인 제작한 모바일 게임입니다.
게임에 사용된 기술 원리를 크게 보면 

1. 카메라가 미리 생성된 포인트를 따라 움직입니다. 

2. 캐릭터가 충격을 받으면 충격 반대 방향으로 후퇴.

3. 도깨비 불이 특정 구역 안에서 랜덤 스폰해서 구역 안에서만 랜덤으로 배회합니다.

4. 도깨비가 나타나 주인공에게 다가가 불을 던집니다.

5. 수직으로 떨어지는 불을 적게 사용하여 효과적으로 연출

6. 보스는 4개의 공격 패턴이 있고 랜덤으로 구현

7. 게임 중 특정 조건에 해당하면 Playerpref에 저장하여 수집록에 표시

1. LetterBox

2. IAP 결제
